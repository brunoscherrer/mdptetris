Detailed Description

Tetris pieces.

This module handles the game pieces. A piece is composed by an array of orientations. Each orientation is an array of rows, where each row is represented by a 16-bit integer.

Creates a set of pieces as described in a given file.

The file describes the existing pieces of the game. The standard Tetris pieces are defined in the file pieces4.dat. The pieces are loaded from the file into the pointer given in parameter.

Parameters
file_namename of the file with the definition of the pieces
nb_piecesthis pointer will receive the number of pieces
