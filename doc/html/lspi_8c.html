◆ NB_GAMES_TO_EVALUATE

#define NB_GAMES_TO_EVALUATE   100

This module executes the least-squares policy iteration algorithm.

You can stop the algorithm with Ctrl-C. The current iteration is then terminated. The policy is saved into the feature based policy file you specified in the command line.

This feature file can be used to continue the execution later or to play in the simulator (./tetris).
