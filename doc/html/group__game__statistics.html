Detailed Description

Statistics about some sequences (episodes) of games played.

This module handles the statistics about one or several sequences of games played. A seqeuence of games is called an episode. Each episode is saved as a line in the statistics file. The line contains the episode number, the best score, the worst score, the mean score, the standard deviation and the scores of the games played during the episode. Optionnaly, it can also contain the feature weights if the games were played with a feature-based policy.

As the first element of each line is the episode number and the second element is the episode's mean score, the file can be plotted directly in Gnuplot.

Ends the current episode.

This function has to be called when all games of the episode are played. The statistics of the episode are printed into a line of the file if it was not NULL. The statistics are then reseted for the next episode, and nb_episodes is incremented.
